# Contenu de la boîte - Grillo One

Cette page décrit tous les composants inclus avec votre capteur Grillo One.

## Contenu du colis

<img src="/img/grillo-one-contents.jpg" alt="Contenu du colis Grillo One" style={{maxWidth: '800px', width: '100%'}} />

### 1. Unité capteur Grillo One

L'appareil capteur principal contenant :
- Accéléromètre MEMS 3 axes haute précision (qualité sismique, plage ±2g)
- Microcontrôleur double cœur
- Connectivité WiFi et Ethernet
- 3 indicateurs LED RGB de statut
- Port USB (pour l'alimentation et les mises à jour du firmware)

### 2. Adaptateur secteur

- Entrée : 100-240V CA
- Sortie : 5V CC
- Pour une alimentation continue

### 3. Câble Ethernet

- Câble standard Cat5e/Cat6
- Connecteurs RJ45
- Pour une connexion réseau filaire (recommandé pour une fiabilité maximale)

### 4. Guide de démarrage rapide

Carte imprimée avec les instructions de base et un code QR vers la documentation en ligne.

### 5. Matériel de montage

- Vis de montage
- Chevilles murales

## Détails de l'unité capteur

<img src="/img/grillo-one-sensor.jpg" alt="Unité capteur Grillo One" style={{maxWidth: '800px', width: '100%'}} />

### LEDs de statut

Le Grillo One dispose de trois LEDs RGB de statut :

| LED | Fonction |
|-----|----------|
| LED Réseau | Indique l'état de connexion (Ethernet/WiFi) |
| LED Capteur | Indique l'état de l'accéléromètre |
| LED Données | Clignote lors de la transmission des données |

### Ports et connecteurs

- **Port Ethernet** - Connecteur RJ45 pour connexion réseau filaire
- **Port USB** - Pour l'alimentation et les mises à jour du firmware
- **Entrée d'alimentation CC** - Pour la connexion de l'adaptateur secteur

### Dessous

- **Trous de montage** — pour la fixation par vis
- **Étiquette d'identification** — nécessaire pour le provisionnement (voir ci-dessous)

## Vérifiez votre colis

Avant de commencer l'installation, vérifiez que tous les composants sont présents et en bon état :

- [ ] L'unité capteur est intacte sans dommage visible
- [ ] L'étiquette d'identification est présente (sur l'appareil ou l'emballage)
- [ ] L'adaptateur secteur et le câble sont présents
- [ ] Le câble Ethernet est inclus
- [ ] Le guide de démarrage rapide est présent
- [ ] Le matériel de montage est complet

:::warning Composants manquants ou endommagés ?
Si des composants sont manquants ou endommagés, [contactez le support Grillo](/support/contact) avant de procéder à l'installation.
:::

## Étiquette d'identification

Chaque capteur possède une étiquette d'**identification** — vous en aurez besoin pour revendiquer et provisionner le capteur dans Grillo Cloud.

L'étiquette se trouve :

1. **Sur l'emballage** — étiquette sur la boîte extérieure
2. **Sur l'appareil** — dessous de l'unité capteur

L'étiquette contient :
- **ID de l'appareil** — identifiant unique requis pour le provisionnement
- **Code QR** — pour un scan rapide pendant l'installation

:::tip
Prenez une photo de l'étiquette d'identification avant l'installation. Vous en aurez besoin lors de l'ajout du capteur à votre tableau de bord Grillo Cloud.
:::
