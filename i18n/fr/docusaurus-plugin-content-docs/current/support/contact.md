### Support par email

Pour les problèmes non couverts dans la documentation :

**Email :** support@grillo.io

Incluez dans votre email :
- Le nom de votre organisation
- Le(s) numéro(s) de série du capteur si applicable
- Description du problème
- Les étapes que vous avez déjà essayées
- Captures d'écran si utile

**Délai de réponse :** Généralement sous 1-2 jours ouvrables

## Avant de contacter le support

### Consulter la documentation

De nombreux problèmes sont couverts dans nos guides :
- [Dépannage Grillo One](/hardware/grillo-one/troubleshooting)
- [Dépannage Grillo Pulse](/hardware/grillo-pulse/troubleshooting)
- [FAQ](/support/faq)

### Rassembler les informations

Aidez-nous à vous aider plus rapidement en collectant :

**Pour les problèmes de capteur :**
- Numéro de série du capteur
- Statut/couleur de la LED
- Type de réseau (WiFi/Ethernet/Cellulaire)
- Ce que vous avez déjà essayé

**Pour les problèmes de tableau de bord :**
- Navigateur et version
- Captures d'écran du problème
- Étapes pour reproduire

**Pour les problèmes de compte :**
- Adresse email du compte
- Nom de l'organisation

### Essayer le dépannage de base

Corrections courantes :
1. Actualiser la page
2. Vider le cache du navigateur
3. Redémarrer le capteur (débrancher pendant 30 secondes)
4. Vérifier la connectivité réseau
5. Essayer un autre navigateur

## Que inclure dans les demandes de support

### Bonne demande de support

> **Sujet :** Le capteur GR001234 apparaît hors ligne depuis 24 heures
>
> Bonjour,
>
> Mon capteur Grillo One (série : GR001234) apparaît hors ligne dans le tableau de bord depuis 24 heures.
>
> Ce que j'ai essayé :
> - Vérifié l'alimentation (LED vert fixe)
> - Vérifié que le WiFi fonctionne (autres appareils connectés)
> - Redémarré le capteur
>
> Le capteur est connecté à un réseau WiFi 2,4 GHz. Rien n'a changé dans la configuration réseau.
>
> Organisation : ACME Research
>
> Merci de m'indiquer les prochaines étapes.

### Pas utile

> Mon capteur ne fonctionne pas. Merci de le réparer.

## Demandes de fonctionnalités

Vous avez une idée pour améliorer Grillo ?

**Email :** feedback@grillo.io

Incluez :
- Description de la fonctionnalité
- Cas d'usage / pourquoi ce serait utile
- Exemples ou maquettes éventuels

Nous examinons toutes les suggestions mais ne pouvons garantir leur implémentation.

## Signalement de bugs

Vous avez trouvé un bug dans le tableau de bord ou la documentation ?

**Email :** support@grillo.io

**Objet :** Rapport de bug - [brève description]

Incluez :
- Ce que vous attendiez
- Ce qui s'est réellement passé
- Étapes pour reproduire
- Navigateur, OS et appareil
- Captures d'écran ou enregistrement d'écran
