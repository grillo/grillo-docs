# Paramètres du profil

Gérez vos informations personnelles et vos préférences dans les paramètres de votre profil.

## Accéder aux paramètres du profil

1. Connectez-vous à [cloud.grillo.io](https://cloud.grillo.io)
2. Cliquez sur votre icône/avatar de profil en haut à droite
3. Sélectionnez **« Profil »** ou **« Paramètres »** dans le menu déroulant

## Informations personnelles

### Nom d'affichage

Votre nom tel qu'il apparaît aux autres utilisateurs de vos organisations.

1. Accédez aux paramètres du profil
2. Trouvez le champ « Nom d'affichage »
3. Entrez votre nom préféré
4. Cliquez sur « Enregistrer » pour mettre à jour

### Adresse email

Votre email principal pour la connexion et les notifications.

:::note
Changer votre email peut nécessiter une nouvelle vérification et pourrait affecter votre méthode de connexion.
:::

Pour changer votre email :
1. Accédez aux paramètres du profil
2. Cliquez sur « Changer l'email »
3. Entrez votre nouvelle adresse email
4. Vérifiez le nouvel email via le lien envoyé
5. Votre compte sera mis à jour

### Photo de profil

Ajouter ou changer votre avatar :

1. Accédez aux paramètres du profil
2. Cliquez sur votre avatar actuel ou « Ajouter une photo »
3. Téléchargez une image (JPG, PNG recommandés)
4. Ajustez le recadrage si nécessaire
5. Enregistrez les modifications

## Préférences de notification

Contrôlez comment et quand vous recevez des notifications :

### Notifications par email

| Type de notification | Description |
|---------------------|-------------|
| Alertes capteur hors ligne | Quand un capteur passe hors ligne |
| Événements sismiques | Quand des événements sont détectés (si activé) |
| Résumé hebdomadaire | Résumé d'activité hebdomadaire |
| Alertes de sécurité | Tentatives de connexion, changements de mot de passe |

Pour configurer :
1. Accédez aux paramètres du profil
2. Trouvez la section « Notifications »
3. Activez/désactivez chaque type de notification
4. Enregistrez les modifications

## Comptes connectés

Voir et gérer les méthodes d'authentification connectées :

### Voir les comptes connectés

1. Accédez aux paramètres du profil
2. Trouvez la section « Comptes connectés »
3. Voyez quelles méthodes sont liées (Google, Microsoft, email)

### Ajouter une connexion

Pour ajouter une méthode de connexion supplémentaire :

1. Cliquez sur « Connecter » à côté du fournisseur
2. Authentifiez-vous avec ce fournisseur
3. Le compte sera lié

### Supprimer une connexion

:::warning
Assurez-vous d'avoir au moins une méthode de connexion avant d'en supprimer d'autres.
:::

1. Cliquez sur « Déconnecter » à côté du fournisseur
2. Confirmez l'action
3. Cette méthode de connexion ne fonctionnera plus

## Données du compte

### Exporter vos données

Pour demander un export de vos données de compte :

1. Accédez aux paramètres du profil
2. Trouvez la section « Données et confidentialité »
3. Cliquez sur « Exporter mes données »
4. Vous recevrez un email avec un lien de téléchargement

### Suppression du compte

Pour supprimer votre compte Grillo :

1. Accédez aux paramètres du profil
2. Trouvez la section « Zone de danger » ou « Supprimer le compte »
3. Cliquez sur « Supprimer le compte »
4. Confirmez en tapant votre email/mot de passe
5. Votre compte sera programmé pour suppression

:::danger
La suppression du compte est permanente et :
- Vous retirera de toutes les organisations
- Supprimera vos paramètres personnels
- Supprimera vos données (après la période de conservation)

Cela ne peut pas être annulé.
:::

## Gestion des sessions

### Voir les sessions actives

Voyez où vous êtes actuellement connecté :

1. Accédez aux paramètres du profil
2. Trouvez la section « Sessions » ou « Appareils »
3. Voir la liste des sessions actives avec :
   - Type d'appareil/navigateur
   - Localisation (approximative)
   - Dernière activité

### Se déconnecter des sessions

Pour vous déconnecter d'une session spécifique :

1. Trouvez la session dans la liste
2. Cliquez sur « Se déconnecter » à côté
3. Cette session sera terminée

Pour vous déconnecter de toutes les sessions :

1. Cliquez sur « Se déconnecter de tous les appareils »
2. Confirmez l'action
3. Vous serez déconnecté partout (y compris l'appareil actuel)

## Dépannage

### Les modifications ne s'enregistrent pas

1. Vérifiez votre connexion Internet
2. Rafraîchissez la page et réessayez
3. Videz le cache du navigateur
4. Essayez un autre navigateur

### Impossible de changer l'email

1. Vérifiez que vous possédez la nouvelle adresse email
2. Vérifiez que le nouvel email n'est pas déjà utilisé
3. Contactez le support si les problèmes persistent

### La photo de profil ne se télécharge pas

1. Vérifiez la taille du fichier (généralement max 5 Mo)
2. Utilisez le format JPG ou PNG
3. Essayez une autre image
4. Essayez un autre navigateur
