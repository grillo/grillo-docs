# Accepter les invitations

Vous pouvez rejoindre des organisations existantes dans Grillo Cloud en acceptant une invitation d'un administrateur d'organisation.

## Comment fonctionnent les invitations

1. Un administrateur d'organisation vous envoie une invitation par email
2. Vous recevez un email avec un lien d'invitation
3. Vous cliquez sur le lien pour accepter
4. Vous rejoignez l'organisation avec le rôle attribué

## Recevoir une invitation

### Email d'invitation

Quand quelqu'un vous invite dans son organisation, vous recevrez un email contenant :

- Le nom de l'organisation
- Qui vous a invité
- Votre rôle attribué
- Un lien pour accepter l'invitation

## Accepter une invitation

### Si vous avez déjà un compte Grillo

1. Cliquez sur le lien **« Accepter l'invitation »** dans l'email
2. Connectez-vous avec vos identifiants existants
3. Vous serez ajouté automatiquement à l'organisation
4. Le tableau de bord affichera votre nouvelle organisation

### Si vous n'avez pas encore de compte

1. Cliquez sur le lien **« Accepter l'invitation »** dans l'email
2. Vous serez invité à créer un compte
3. Choisissez votre méthode d'inscription (Google, Microsoft ou email)
4. Complétez le processus de création de compte
5. Vous serez automatiquement ajouté à l'organisation

## Après avoir accepté

Une fois que vous avez accepté une invitation :

### Vérifiez votre rôle

Votre rôle détermine ce que vous pouvez faire dans l'organisation :

| Rôle | Capacités |
|------|-----------|
| Lecteur | Voir les capteurs et les données |
| Membre | Ajouter des capteurs, modifier les configurations |
| Admin | Gérer les membres, créer des réseaux |
| Propriétaire | Contrôle total, facturation, supprimer l'org |

[En savoir plus sur les rôles →](/dashboard/organizations/user-roles)

### Basculer entre les organisations

Si vous appartenez à plusieurs organisations :

1. Cliquez sur le nom de l'organisation dans la navigation
2. Sélectionnez l'organisation que vous souhaitez voir
3. Le tableau de bord changera de contexte

## Gérer les invitations

### Invitations en attente

Vous pouvez avoir des invitations en attente si :
- Vous avez reçu une invitation mais ne l'avez pas acceptée
- Le lien d'invitation a expiré

Pour voir les invitations en attente :
1. Connectez-vous à votre compte
2. Recherchez une icône de notifications ou une section « Invitations »
3. Acceptez ou refusez les invitations en attente

### Refuser une invitation

Si vous ne souhaitez pas rejoindre une organisation :
- Ne cliquez simplement pas sur le lien d'acceptation
- Ou recherchez une option « Refuser » dans l'email
- L'invitation finira par expirer

## Dépannage

### Le lien d'invitation ne fonctionne pas

1. **Lien expiré** - Demandez à l'expéditeur d'envoyer une nouvelle invitation
2. **Déjà acceptée** - Essayez de vous connecter ; vous êtes peut-être déjà membre
3. **Mauvais email** - Assurez-vous d'être connecté avec l'email invité

### Mauvais rôle attribué

Si on vous a attribué le mauvais rôle :
1. Contactez l'administrateur de l'organisation
2. Il peut changer votre rôle dans les paramètres des membres
3. [Voir Gérer les membres](/dashboard/organizations/managing-members)

### Impossible de trouver l'organisation

Après avoir accepté, si vous ne voyez pas l'organisation :
1. Rafraîchissez la page
2. Vérifiez le menu déroulant de changement d'organisation
3. Déconnectez-vous et reconnectez-vous
4. Contactez l'administrateur qui vous a invité

### Invité au mauvais email

Si l'invitation a été envoyée à la mauvaise adresse email :
1. Contactez l'administrateur de l'organisation
2. Demandez-lui d'annuler l'ancienne invitation
3. Demandez une nouvelle invitation à votre email correct

## Quitter une organisation

Si vous souhaitez quitter une organisation que vous avez rejointe :

1. Accédez aux paramètres de l'organisation
2. Recherchez l'option « Quitter l'organisation »
3. Confirmez votre choix

:::warning
Quitter une organisation supprime votre accès à tous les réseaux et données de cette organisation.
:::
