# Créer un compte

Pour utiliser Grillo Cloud, vous devez créer un compte sur [cloud.grillo.io](https://cloud.grillo.io).

## Options d'inscription

Grillo Cloud prend en charge plusieurs méthodes d'authentification :

| Méthode | Idéal pour |
|---------|------------|
| **Google** | Inscription rapide si vous avez un compte Google |
| **Microsoft** | Organisations utilisant Microsoft 365 |
| **Email** | Quand vous préférez des identifiants séparés |

## S'inscrire avec Google

1. Allez sur [cloud.grillo.io](https://cloud.grillo.io)
2. Cliquez sur **« S'inscrire »** ou **« Commencer »**
3. Cliquez sur **« Continuer avec Google »**
4. Sélectionnez votre compte Google ou connectez-vous
5. Accordez à Grillo les autorisations demandées
6. Complétez votre profil si demandé

## S'inscrire par email

1. Allez sur [cloud.grillo.io](https://cloud.grillo.io)
2. Cliquez sur **« S'inscrire »** ou **« Commencer »**
3. Cliquez sur **« S'inscrire par email »**
4. Entrez votre adresse email
5. Créez un mot de passe fort
6. Cliquez sur **« Créer le compte »**
7. Vérifiez votre email pour le lien de vérification
8. Cliquez sur le lien de vérification pour activer votre compte

### Exigences du mot de passe

<!-- TODO: Confirm actual password requirements -->

- Minimum 8 caractères
- Au moins une lettre majuscule
- Au moins une lettre minuscule
- Au moins un chiffre
- Au moins un caractère spécial

## Vérification de l'email

Si vous vous inscrivez par email :

1. Vérifiez votre boîte de réception pour un email de Grillo
2. Cliquez sur le lien de vérification dans l'email
3. Votre compte sera activé

:::note
Les emails de vérification peuvent prendre quelques minutes à arriver. Vérifiez votre dossier spam si vous ne le voyez pas.
:::

### Renvoyer l'email de vérification

Si vous n'avez pas reçu l'email de vérification :

1. Essayez de vous connecter avec vos identifiants
2. Recherchez l'option « Renvoyer l'email de vérification »
3. Cliquez pour demander un nouvel email
4. Vérifiez votre boîte de réception (et le dossier spam)

## Après l'inscription

Une fois votre compte créé :

### Configuration initiale

1. **Complétez votre profil** - Ajoutez votre nom et vos coordonnées
2. **Créez ou rejoignez une organisation** - [Voir le guide des organisations](/dashboard/organizations)
3. **Configurez votre premier réseau** - [Voir le guide des réseaux](/dashboard/networks)

### Si vous avez été invité

Si quelqu'un vous a invité dans son organisation :

1. Vous pouvez recevoir un email d'invitation avant de vous inscrire
2. Cliquez sur le lien dans l'email d'invitation
3. Créez votre compte (ou connectez-vous si vous en avez déjà un)
4. Vous serez automatiquement ajouté à l'organisation

[En savoir plus sur l'acceptation des invitations →](/dashboard/account/accepting-invites)

## Comptes multiples

### Puis-je avoir plusieurs comptes ?

Chaque adresse email ne peut être associée qu'à un seul compte Grillo. Cependant :

- Vous pouvez appartenir à plusieurs organisations avec un seul compte
- Les connexions sociales (Google/Microsoft) créent des comptes liés à ces emails

### Changer d'organisation

Si vous êtes membre de plusieurs organisations :

1. Cliquez sur le nom de votre organisation dans la navigation
2. Sélectionnez une autre organisation dans le menu déroulant
3. Le tableau de bord basculera vers la vue de cette organisation

## Dépannage

### « Email déjà enregistré »

Cet email est déjà associé à un compte :

1. Essayez de vous connecter au lieu de vous inscrire
2. Utilisez « Mot de passe oublié » si nécessaire
3. Si vous avez utilisé Google/Microsoft avant, essayez cette méthode

### Impossible de se connecter avec Google/Microsoft

1. Assurez-vous d'utiliser le même compte avec lequel vous vous êtes inscrit
2. Essayez de vider le cache de votre navigateur
3. Vérifiez que les cookies tiers sont activés
4. Essayez un autre navigateur

### Compte verrouillé

Après trop de tentatives de connexion échouées :

1. Attendez 15-30 minutes avant de réessayer
2. Utilisez « Mot de passe oublié » pour réinitialiser vos identifiants
3. Contactez le support si le problème persiste
