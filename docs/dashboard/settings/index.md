# Dashboard Settings

Configure your Grillo Cloud dashboard preferences, notifications, and integrations.

## Accessing settings

1. Click the gear icon in the navigation
2. Or navigate to Profile → Settings
3. Choose the settings category

## Settings categories

### Notifications

Configure how and when you receive alerts:
- Email notifications
- Dashboard notifications
- Alert thresholds
- Notification schedules

[Notifications guide →](/dashboard/settings/notifications)

### API Keys

Manage API access for integrations:
- Generate API keys
- View existing keys
- Revoke access
- Set permissions

[API keys guide →](/dashboard/settings/api-keys)

### Preferences

Personal dashboard preferences:
- Timezone
- Language
- Units (metric/imperial)
- Default views

## Organization vs personal settings

| Setting type | Scope | Who can change |
|--------------|-------|----------------|
| Personal | Your account only | You |
| Organization | All org members | Admin/Owner |
| Network | Specific network | Admin/Owner |

### Personal settings

Affect only your experience:
- Timezone display
- Notification preferences
- Dashboard layout
- Language

### Organization settings

Affect all organization members:
- Default configurations
- API access policies
- Billing information
- Security policies

### Network settings

Specific to individual networks:
- Event detection parameters
- Alert rules
- Data retention
- Access controls

## Common settings tasks

| Task | Location |
|------|----------|
| Change timezone | Profile → Preferences |
| Enable email alerts | Settings → Notifications |
| Generate API key | Settings → API Keys |
| Update billing | Organization → Billing |
