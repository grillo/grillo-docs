### What's not covered

## Making a warranty claim

### Step 1: Contact support

Before returning any product:

1. Email support@grillo.io
2. Include:
   - Serial number
   - Purchase date/proof of purchase
   - Description of the problem
   - Photos if applicable

### Step 2: Troubleshooting

Our support team will help troubleshoot to determine if the issue is covered under warranty or can be resolved without replacement.

### Step 3: Return authorization

If replacement is needed:
1. We'll issue a Return Merchandise Authorization (RMA)
2. You'll receive return shipping instructions
3. Return the product in original or equivalent packaging

### Step 4: Replacement

Upon receiving and inspecting the returned product:
- If defect is confirmed, we'll ship a replacement
- If not covered by warranty, we'll contact you with options
