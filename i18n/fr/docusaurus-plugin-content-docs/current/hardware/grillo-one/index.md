# Grillo One

Le Grillo One est un capteur de mouvements forts conçu pour les installations intérieures fixes. Il utilise un accéléromètre MEMS à faible bruit pour capturer les vibrations et les secousses du sol, des événements faibles aux événements forts.

<img src="/img/grillo-one-case.jpg" alt="Grillo One" style={{maxWidth: '800px', width: '100%'}} />

## Caractéristiques principales

- **Accéléromètre MEMS à faible bruit** - 3 canaux de données sismiques
- **Mouvements faibles à forts** - Capture les vibrations et les secousses sismiques
- **Double connectivité** - Options WiFi (2,4 GHz) et Ethernet
- **Streaming en temps réel** - Données continues vers Grillo Cloud
- **Design compact** - Format discret pour les espaces intérieurs

## Spécifications

### Physiques

| Spécification | Valeur |
|---------------|--------|
| Dimensions | 50 x 80 mm |
| Poids | 20g |
| Boîtier | Impression 3D (boîtier extérieur IP65 en option) |
| Température de fonctionnement | -20°C à +60°C |

### Alimentation

| Spécification | Valeur |
|---------------|--------|
| Source d'alimentation | Adaptateur secteur (5V CC) |
| Consommation | 1,32W (400mA à 3,3V) |
| USB | Alimentation et mises à jour du firmware |

### Connectivité

| Spécification | Valeur |
|---------------|--------|
| Connexion principale | Ethernet |
| Connexion secondaire | WiFi 802.11 b/g/n (2,4 GHz) |
| Transmission des données | Temps réel |
| Synchronisation horaire | NTP |

### Capteur

| Spécification | Valeur |
|---------------|--------|
| Type de capteur | Accéléromètre triaxial MEMS (ADC 20 bits) |
| Canaux | 3 |
| Densité de bruit | 22,5 µg/√Hz |
| Plage de mesure | ±2g |
| Fréquence d'échantillonnage | 125 Hz |
| Plage de détection | Mouvements du sol faibles à forts |

## Aperçu de l'installation

1. **Déballage** - Vérifier que tous les composants sont présents
2. **Installation** - Monter le capteur dans un emplacement approprié
3. **Connexion** - Configurer le WiFi ou brancher l'Ethernet
4. **Provisionnement** - Enregistrer le capteur sur Grillo Cloud
5. **Configuration** - Définir le nom de la station et l'emplacement dans le tableau de bord

**Temps d'installation estimé :** 15-20 minutes

## Prérequis

Avant de commencer, assurez-vous d'avoir :

- [ ] Un compte Grillo Cloud ([créez-en un ici](https://cloud.grillo.io))
- [ ] Une organisation et un réseau configurés dans le tableau de bord
- [ ] Les identifiants du réseau WiFi ou une connexion Ethernet
- [ ] Une prise de courant près de l'emplacement d'installation
- [ ] Un emplacement de montage approprié (surface stable, éloignée des sources de vibration)
